# E-Commerce Dashboard using Power BI

## Project Overview
The E-Commerce Dashboard is an interactive business analytics project developed using Power BI to analyze sales, profit, orders, and shipment performance. The dashboard helps in understanding business trends and supports data-driven decision-making through visual insights.

## Objectives
- Analyze overall sales and profit performance
- Monitor shipment and order status
- Identify sales trends and customer insights
- Create interactive and user-friendly business reports

## Tools & Technologies Used
- Power BI
- DAX
- Excel / CSV Dataset
- Data Visualization

## Dashboard Features
- KPI Cards for Sales, Profit, and Orders
- Interactive Filters and Slicers
- Shipment Performance Analysis
- Sales Trend Visualization
- Category and Customer Insights
- Dynamic Business Reporting

## Key Insights
- Tracked overall business revenue and profit trends
- Analyzed shipment performance and delivery insights
- Identified high-performing sales categories
- Visualized customer and order behavior effectively
